## ğŸ“¦ **X-Ray Core Concepts Recap**

| ğŸ”¹ Concept      | ğŸ’¡ Description                                    |
| --------------- | ------------------------------------------------- |
| **Segment**     | A unit of work (API call, Lambda, etc.)           |
| **Subsegment**  | A breakdown within a segment (e.g., DB query)     |
